## Script template for Network Serializers in Mirror (Unity 3D)

### Setup:
  1) Add file to Assets > ScriptTemplates
  2) Relaunch Editor

### Usage
  1) Create a new script using Assets (or right-click Project Panel) >  Create > Mirror > Network Serializer
  2) Name Script
